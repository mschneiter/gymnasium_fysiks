Cuando uno hace cambios en archivos locales uno puede empujar los cambios
en el archivo indicado a partir de 

git add main.tex

luego de hacer el add de los archivos es bueno hacer un commit, que
escribe mensajes con los cambios realizados

commit -m "texto que explica los cambios realizados en el archivo"

Hasta este punto solo hemos realizado modificaciones locales, para
empujarlas al servidor tenemos que hacer un push

git push
